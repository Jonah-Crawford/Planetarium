# Planetarium
A html webpage using Craw's SDAPI to show all Stardance Projects.

![Planetarium_Image](https://www.dropbox.com/scl/fi/q6x6l2rff8yq7x7z4x60v/Screenshot-2026-06-16-111550.png?rlkey=jla3p8pafl5alrjswlf6gg69l&st=wd7mtpq2&raw=1)

[Use it here!](https://planetarium.ultimatecraw.xyz/)

## Functions
- Full database searching (slow, but no data lost)
- History tracking, useful for fraud tracking
- Leaderboard stats
- Growth ratings
- Frequent updates (every ~6 hours)

## How it works
The leaderboard uses **Craw's Stardance API** from *CRAW SYSTEMS*, which captures a SQL table of all project data, and then parses it into a html website.
